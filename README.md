# Customization

### Colors

Edit the CSS variables in the root selector:

```css
:root {
  --primary-color: #ec6c3b;
  --secondary-color: #2a2a2a;
  /* ... other colors */
}
```

### Typography

Modify font sizes and weights in the relevant CSS classes

### Images

Replace image paths in HTML with your own product images
