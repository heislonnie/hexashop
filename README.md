Hexashop

Hexashop is a responsive e-commerce website built with HTML, CSS and JavaScript.

The project focuses on layout structure, responsiveness, and basic front-end interactions rather than backend functionality.

This was built as a front-end project to practice modern CSS layout techniques and UI behavior.

Overview

The site includes multiple product sections (Men, Women, Kids), a featured explore section, and a newsletter signup area.
All pages are implemented as a single static site and run entirely in the browser.

There are no external libraries or frameworks used.

Features

Responsive layout for desktop, tablet, and mobile

Structured product sections with reusable card layouts

Hover and scroll interactions

Simple add-to-cart visual feedback

Newsletter subscription form

Semantic HTML for accessibility

CSS optimized for readability and maintainability

Project Structure
Hexashop/
├── index.html
├── hexa.css
├── images and assets
└── README.md

The site is served directly from index.html with styles defined in hexa.css.

Responsive Behavior

The layout adjusts at common breakpoints:

Desktop (1024px and up)

Tablet (768px to 1023px)

Mobile (below 768px)

Flexbox and CSS Grid are used to handle layout changes.

How to Run
Clone or download the repository

Open index.html in a browser

No build step or dependencies are required.

Customization

Colors and spacing can be adjusted directly in the CSS file

Images can be replaced by swapping files in the project directory

Text content can be edited in index.html

Limitations

This project does not include:

Backend logic

Real shopping cart functionality

Authentication or payments

These features would require a server-side implementation.
